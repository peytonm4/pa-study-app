Execute(Guid documentId) implementation:
1. Load document from DB (include Module for path construction)
2. Download source file from S3 to a MemoryStream using storage.DownloadAsync
3. Write file bytes to a temp path in Path.GetTempPath() (needed by Python script as a real file)
4. Build input JSON: `{"document_id": "...", "file_path": "...", "file_type": "pptx|pdf"}`
5. Determine scriptPath: Path.Combine(AppContext.BaseDirectory, "..", "..", "..", "..", "skills", "lecture-extractor-extracted", "extract_images.py") or use a configurable base path from IConfiguration. Keep simple for now: resolve relative to the repo root by walking up from BaseDirectory, or accept as an injected IConfiguration value at key "Skills:BasePath". Use IConfiguration with key "Skills:BasePath" (default: "src/skills") so it's overridable in tests.
6. Call `var manifestJson = await skillRunner.RunAsync(scriptPath, inputJson, ct)`
7. Deserialize manifestJson to `FigureManifest` record: `record FigureManifest(List<FigureEntry> Figures)` where `FigureEntry` has `string Id, string S3Key, int Page, bool HasCaption, string? LabelType`
8. For each entry: create Figure entity with Keep = entry.HasCaption, PageNumber = entry.Page, LabelType = entry.LabelType, S3Key = entry.S3Key, ManifestMetadataJson = JsonSerializer.Serialize(entry)
9. db.Figures.AddRange(figures); await db.SaveChangesAsync()
10. For each kept figure: call visionProvider.ExtractTextAsync to get caption — download image bytes first via storage.DownloadAsync, update figure.Caption, save again
11. Clean up temp file in finally block
