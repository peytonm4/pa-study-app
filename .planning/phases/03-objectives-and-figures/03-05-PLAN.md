Create LectureExtractionJob.cs in src/Api/Jobs/. Constructor takes (AppDbContext db, ISkillRunner skillRunner, IStorageService storage, IConfiguration config). Use [AutomaticRetry(Attempts = 1)].

Execute(Guid moduleId) implementation:
1. Load module from DB, set ExtractionStatus = Processing, SaveChanges
2. Fetch all Keep=true Figures for this module (via Document.ModuleId join)
3. Fetch all Chunks for this module (text content for the skill)
4. Build inputJson: `{"module_id": "...", "chunks": [{...}], "figures": [{...}]}`
5. Determine scriptPath via config["Skills:BasePath"] + "/lecture-extractor-extracted/lecture_extractor.py"
6. Call `var resultJson = await skillRunner.RunAsync(scriptPath, inputJson, ct)`
7. Deserialize resultJson to `LectureResult` record with `List<SectionDto> Sections` — SectionDto has int Level, string Heading, string Content, List<int> Pages, List<string> Figures
8. Map to Section entities using SortOrder = index, SourcePageRefsJson = JsonSerializer.Serialize(dto.Pages)
9. db.Sections.AddRange(sections); await db.SaveChangesAsync()
10. Build .docx using DocumentFormat.OpenXml with MemoryStream:
    - Create WordprocessingDocument.Create(ms, WordprocessingDocumentType.Document)
    - Add MainDocumentPart, set Document with Body
    - Add StyleDefinitionsPart with minimal Heading1/Heading2/Heading3 styles (use Name/StyleId pairs: "heading 1"/"Heading1", etc.) to prevent plain-text rendering in Word
    - For each section: add Paragraph with ParagraphStyleId = "Heading1/2/3" based on HeadingLevel, then add content Paragraph
    - Save document, reset ms.Position = 0
11. Upload .docx: `var s3Key = $"modules/{moduleId}/lecture.docx"`, call storage.UploadAsync(ms, s3Key, "application/vnd.openxmlformats-officedocument.wordprocessingml.document")
12. Set module.DocxS3Key = s3Key, module.ExtractionStatus = Ready, SaveChanges

Wrap steps 1-12 in try/catch: on exception, set module.ExtractionStatus = Failed, module.ExtractionError = ex.Message, SaveChanges, then rethrow.

Add internal record types LectureResult and SectionDto in same file for JSON deserialization.
