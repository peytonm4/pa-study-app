public class IngestionJob(

        document.Status = DocumentStatus.Processing;
        await db.SaveChangesAsync();

        // Download from S3 using document.S3Key
        // Extract chunks based on document.ContentType
        // Save chunks to db
        // Fan out vision jobs for blank PDF pages
        // Update document.Status

        var flaggedPages = chunks.Where(c => c.IsVisionExtracted == false && needsVision).ToList();
        document.PendingVisionJobs = flaggedPages.Count;
        document.Status = flaggedPages.Count > 0 ? DocumentStatus.Processing : DocumentStatus.Ready;

var remaining = await db.Documents

Create `src/Worker/Jobs/IngestionJob.cs`:
```
Constructor injects: IBackgroundJobClient, IPptxExtractor, IPdfExtractor, IStorageService, AppDbContext

Execute(Guid documentId):
1. Load document from DB — throw if not found
2. Set document.Status = Processing, save
3. Download file stream via IStorageService.DownloadAsync(document.S3Key)
4. Copy stream to MemoryStream (avoid stream disposal issues with OpenXml/PdfPig)
5. Dispatch by content type:
   - PPTX: call pptxExtractor.Extract(memStream, document.FileName)
     → create one Chunk per SlideContent: { DocumentId, FileName=slide.FileName, PageNumber=slide.SlideNumber, Content=slide.BodyText + "\n" + slide.NotesText, IsVisionExtracted=false }
     → no vision jobs for PPTX
   - PDF: call pdfExtractor.Extract(memStream, document.FileName)
     → create one Chunk per PageContent: { DocumentId, FileName=page.FileName, PageNumber=page.PageNumber, Content=page.Text, IsVisionExtracted=page.NeedsVision }
     → collect pages where NeedsVision=true as flaggedPages
   - Unknown content type: set document.Status=Failed, document.ErrorMessage="Unsupported file type", save, return
6. Save all chunks to DB via db.Chunks.AddRange(chunks); await db.SaveChangesAsync()
7. Set document.PendingVisionJobs = flaggedPages.Count
8. Set document.Status = flaggedPages.Count > 0 ? DocumentStatus.Processing : DocumentStatus.Ready
9. Save document
10. Enqueue VisionExtractionJob for each flagged page
```

```
Constructor injects: IVisionProvider, IStorageService, AppDbContext

[AutomaticRetry(Attempts = 3)]
Execute(Guid documentId, int pageNumber):
1. Find chunk: db.Chunks.FirstOrDefaultAsync(c => c.DocumentId == documentId && c.PageNumber == pageNumber)
   → if not found, log warning and return (idempotent)
2. Find document: db.Documents.FindAsync(documentId)
3. Download S3 file using document.S3Key → get stream
4. For vision extraction, the file is a PDF. We need the raw PDF page bytes.
   The current IVisionProvider interface takes byte[] imageBytes and string mimeType.
   For stub mode: call visionProvider.ExtractTextAsync(Array.Empty<byte>(), "application/pdf")
   — StubVisionProvider ignores the bytes and returns placeholder text.
   For real Gemini: download full PDF bytes, pass with mimeType="application/pdf"
   (Gemini can accept PDF bytes directly per research open question recommendation).
5. chunk.Content = extracted text from vision provider
6. chunk.IsVisionExtracted = true
7. await db.SaveChangesAsync()
8. Atomic decrement of PendingVisionJobs (use ExecuteUpdateAsync):
   await db.Documents
       .Where(d => d.Id == documentId)
       .ExecuteUpdateAsync(s => s.SetProperty(d => d.PendingVisionJobs, d => d.PendingVisionJobs - 1));
9. Re-fetch document; if PendingVisionJobs == 0 → set Status=Ready, save
```
